# Frost Damage Image Classification

## Overview
This Jupyter notebook demonstrates the process of data preprocessing, augmentation, and training convolutional neural networks (CNN) to classify frost damage in images. It includes steps for loading data, applying data augmentations, building and training a CNN from scratch, evaluating model performance, and setting up transfer learning workflows.

## Features
- Download and unzip dataset from an external source
- Data loading and preprocessing with TensorFlow datasets
- Image augmentation for better generalization
- Define and train a custom CNN model with TensorFlow Keras
- Evaluate model using precision, recall, and F1-score metrics
- Demonstration of transfer learning setup
- Compatible with Google Colab and local Jupyter environments

## Tech Stack
- Python 3
- Jupyter Notebook / Google Colab
- TensorFlow and Keras
- scikit-learn (metrics)
- NumPy, Pandas, Matplotlib
- tqdm, OpenCV, PIL

## How to Use
1. Clone this repository.
2. Upload your data or mount Google Drive if using Colab.
3. Run cells sequentially to preprocess data and train models.
4. Adjust parameters and augmentations as needed for your dataset.
5. Evaluate the trained model performance on test data.
